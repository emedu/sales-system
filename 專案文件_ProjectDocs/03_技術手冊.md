# 03_技術手冊

## 1. 系統架構說明
本系統採用 **Next.js Full Stack** 架構：
- **Frontend**: React Server Components (RSC) + Client Components (互動表單)。
- **Backend**: Next.js App Router API Routes。
- **Database**: SQLite (位於專案根目錄 `dev.db`)。

### 目錄結構
- `src/app`: 頁面路由
  - `sales/`: 成交回報頁面
  - `dashboard/`: 報表頁面
  - `api/`: 後端接口
- `src/components`: UI 元件 (Shadcn/UI)
- `prisma`: 資料庫定義與種子資料

## 2. 安裝與啟動指南 (Installation & Setup)
### 前置需求
- Node.js LTS (v18+)
- Git

### 首次安裝
```bash
# 1. 進入專案目錄
cd sales-system

# 2. 安裝依賴
npm install

# 3. 初始化資料庫
npx prisma db push
npx tsx prisma/seed.ts # 寫入預設資料
```

### 啟動開發伺服器
```bash
npm run dev
```

## 3. API 說明
| 方法 | 路徑 | 描述 |
| --- | --- | --- |
| GET | `/api/students` | 取得所有學生資料 |
| GET | `/api/products` | 取得所有課程資料 |
| POST | `/api/sales` | 新增成交紀錄 (需帶入 studentId, productId) |
| GET | `/api/dashboard` | 取得戰情分析所需之聚合數據 |
