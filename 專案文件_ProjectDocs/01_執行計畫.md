# 01_執行計畫

## 1. 專案目標
將原有的 Google Sheet + AppSheet 洽詢人數成交回報流程，遷移至自主託管的 Modern Web App，以解決資料同步不穩、跨年度資料處理困難以及介面操作體驗不佳的問題。

## 2. 開發步驟規劃
- [x] **Phase 1: 需求分析與架構設計** (已完成)
- [x] **Phase 2: 專案環境建置** (Next.js + Tailwind + SQLite) (已完成)
- [x] **Phase 3: 後端 API 開發** (Prisma Schema, CRUD, Dashboard Aggregation) (已完成)
- [x] **Phase 4: 前端介面開發** (Sales Form, Dashboard UI) (已完成)
- [x] **Phase 5: 驗證與交付** (已完成)

## 3. 使用技術棧
- **核心框架**: Next.js 14 (App Router)
- **程式語言**: TypeScript
- **樣式庫**: Tailwind CSS + Shadcn/UI
- **資料庫**: SQLite (檔案型資料庫)
- **ORM**: Prisma
- **通知元件**: Sonner
